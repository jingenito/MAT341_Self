 git add [file_name]               stage a file to the staging area
 git commit -m "Message"           commit staged files with the specified message

 git push                          push to Github (Good practice to ALWAYS pull before pushing)

 git stash pop                     pop from the stash
 git stash list                    show a list of all stashes 
 git status                        show the current status of the repo
